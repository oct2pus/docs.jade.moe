Other Commands

Summary

JadeBot has a lot of simple commands that don’t require any input from you. Here is a list of them and what they do.

Usage

About

==>about
==>credits

About displays a short accreditation blurb for JadeBot. You can also call it from credits.

Discord

==>discord

Discord displays a link to the JadeBot discord server.

Invite

==>invite

Invite displays a code to invite JadeBot to your discord server.

Pester

==>pester
==>markov

Pester is an alternative to calling an empty ==> to display a fake pesterlog built from Jade’s dialog.

Note: This command may not exist if someone is hosting a 3rd party JadeBot.
