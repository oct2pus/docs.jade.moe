Tips & Tricks

Command Correction

JadeBot now takes commands ==>LikeThis like most other discord bots, but she displays a helpful message if you accidentally write it ==> LikeThis.

Fake Pesterlogs

When you enter an empty ==>, JadeBot will say a line randomly generated using all of Jade’s Pesterlogs.

Responses

JadeBot has a bunch of cute responses to various phrases in any message she can read. This cannot currently be turned off, but eventually a server admin may opt-out their server from her responses.
